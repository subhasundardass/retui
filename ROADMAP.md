# Retui Roadmap

> **Note:** This roadmap represents the long-term vision for Retui. Priorities may change as the project evolves and community feedback is received.

---

# Vision

Retui aims to be a modern, high-performance Terminal UI framework for Go that enables developers to build desktop-like terminal applications using reusable widgets, window management, and an intuitive development experience.

---

# Current Status

- ✅ Core rendering engine
- ✅ Layout system
- ✅ Window management
- ✅ Modal windows
- ✅ Keyboard navigation
- ✅ Basic widgets
